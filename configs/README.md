# Configs

YAML files that centralize tasks, camera presets, training defaults, and inference logging schema.

- `tasks.yaml` — benchmark task definitions + robustness conditions.
- `camera_profiles.yaml` — reusable camera presets.
- `act_defaults.yaml` — ACT training defaults.
- `smolvla_defaults.yaml` — SmolVLA training defaults.
- `inference_defaults.yaml` — standard inference settings.
- `csv_schema.yaml` — column order/types for inference CSV logs.
- `metrics_flags.yaml` — manual prompt rules for failure modes.
