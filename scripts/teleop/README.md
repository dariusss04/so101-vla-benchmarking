# Teleoperation

Scripts for teleoperating the SO101 robot (with and without cameras).

- `so101_teleop_with_cameras.py` / `.sh` — teleop with front + wrist cameras.
- `so101_teleop_no_cameras.py` / `.sh` — teleop without camera streams.

All parameters (ports, IDs, camera settings, dataset options) are provided via environment variables.
